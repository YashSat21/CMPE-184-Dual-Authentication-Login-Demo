Login Site with Dual Authenticator Support

The demonstration site shows one TOTP secret for Google Authenticator and a second TOTP secret for Microsoft Authenticator. It then lets the user sign in with either Google Authenticator or Microsoft Authenticator respectively.

Important: Google Authenticator and Microsoft Authenticator both support standard TOTP, so in real systems either app can usually scan either QR code. This page labels one QR for each appllication to verify the setup for each app.

  • Open HTML file in VS Code and run it with Live Server / browser.
  • Click Generate setup to create the two QR codes.
  • Enter the username & password, and then provide either the Google/Microsoft code to simulate the login.
